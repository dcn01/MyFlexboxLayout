# MyFlexboxLayout
FlexboxLayout弹性布局中所支持的属性

* __flexDirection__
  * flexDirection 属性决定主轴（和副轴，垂直于主轴）的方向（即项目中子元素的排列方向）；
  有以下四个值:
    * row (default)：主轴为水平方向，起点在左端；
    * row_reverse：主轴为水平方向，起点在右端；
    * column：主轴为垂直方向，起点在上沿；
    * column_reverse：主轴为垂直方向，起点在下沿；
    * 其它属性默默探索吧！
